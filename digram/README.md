
### Pin on ultrasonic

| ultrasonic | esp32   | NODEMCU |
| ---------- | ------- | ------- |
| ECHO       | GPIO 13 | D13     |
| TRIG       | GPIO 12 | D12     |
| VCC        | 3V      | 3V      |
| GND        | GND     | GND     |
